## Análisis de Inteligencia de Negocios: Transacciones Retail en Istanbul (2021–2023)

### Descripción del Proyecto
Este proyecto aplica los conceptos fundamentales de Inteligencia de Negocios (Business Intelligence) para analizar datos de transacciones realizadas en centros comerciales de Istanbul entre 2021 y 2023.
El objetivo es transformar datos transaccionales en información útil para la toma de decisiones estratégicas.

### Objetivo de Negocio
Identificar patrones de compra, tendencias por categoría, comportamiento por grupo demográfico y métricas clave de desempeño que puedan apoyar decisiones comerciales.
